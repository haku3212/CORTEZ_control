# Manual de Usuario

## Registrar una trabajadora

Abra Trabajadoras, complete nombre, tipo de trabajo, precios por kilogramo y fecha de ingreso. Use Guardar trabajadora. El sistema genera el codigo `TRA-001`, `TRA-002` y siguientes.

## Crear un lote

Abra Lotes, seleccione empresa y servicio, registre fecha, personas, peso bruto, peso de envases y bolsas. El peso neto se calcula al guardar. El codigo se genera como `SA-Q-001` o `HB-R-001`.

## Registrar un retiro de empresa

Abra Retiros y registre la empresa, el destino del trabajo, quien entrega, quien recoge, transporte, bolsas y pesos. Use este modulo para controlar lo que se saca de San Agustin o Hermanos Blacut antes de procesarlo.

## Entregar almendra

Abra Entregas, elija un lote y una trabajadora compatible. Registre peso bruto, envase, plazo y responsable. El sistema valida que no se entregue mas que el saldo disponible.

## Registrar una devolucion parcial

Abra Recepciones, seleccione una entrega pendiente y registre los pesos clasificados. Deje sin marcar Recepcion final. La entrega queda como devolucion parcial si todavia existe saldo.

## Registrar una devolucion final

Abra Recepciones, seleccione la entrega, registre la clasificacion y marque Recepcion final. Si queda diferencia, escriba una observacion. La entrega queda terminada.

## Ver porcentajes

Use la ficha de lote o la ficha de trabajadora para revisar rendimiento, producto bueno, descarte, pendientes y productividad calculados desde la base.

## Editar precios por categoria

Abra Precios, elija una categoria y pulse Editar. Cambie el precio Bs/kg, por ejemplo almendra buena 3 Bs/kg o podrida 0,50 Bs/kg. Las recepciones nuevas usaran esos valores para calcular el pago total.

## Comparar trabajadoras

Abra Rendimiento. La tabla muestra trabajos terminados, kilogramos procesados, rendimiento ponderado, productividad y atrasos. Compare quebrado y recorte segun el tipo de trabajo de cada persona.

## Crear un respaldo

Abra Respaldos y pulse Crear copia manual. La pantalla muestra la ruta de la base y la carpeta donde se guardan los respaldos.
